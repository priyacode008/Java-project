# ATM Management System

## Description
ATM Management System is a basic console-based Java application that simulates simple ATM operations. The project allows users to log in using a PIN and perform basic banking activities through a menu-driven interface.

## Features
- PIN verification
- Check balance
- Deposit money
- Withdraw money
- Exit option

## Technologies Used
- Java
- Scanner Class
- if-else statements
- switch case
- while loop
  
## Concepts Used
- Variables
- Conditional statements
- Loops
- User input handling

## Learning Outcome
This project helped in understanding Java basics and menu-driven program implementation.
